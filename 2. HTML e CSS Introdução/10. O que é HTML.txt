
Resumo da Introdução ao HTML:

- O que é HTML:
  - HTML é uma linguagem de marcação, não de programação.
  - Usada para estruturar páginas web, adicionando elementos como texto, imagens, listas e formulários.
  - O esqueleto de qualquer aplicação web, incluindo React, tem HTML por trás.

- Características do HTML:
  - Não requer compilação; basta um editor de texto para escrever o código e um navegador para visualizar.
  - Acessível e fácil de aprender.

- Demonstrando HTML na Prática:
  - Qualquer página web pode ter seu HTML inspecionado via ferramentas de desenvolvimento do navegador.
  - Elementos como <h2> (título) e <p> (parágrafo) formam a estrutura visual da página.
  - CSS é essencial para estilizar e polir a estrutura HTML.
