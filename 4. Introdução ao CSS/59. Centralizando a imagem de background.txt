ResumoNesta aula, aprendemos a técnica de centralizar a imagem de background. Essa técnica é útil para melhorar a visualização e aproveitar o espaço útil da tela.Destaques📷 Centralizar a imagem de background é útil para melhorar a visualização e aproveitar o espaço útil da tela.🖼️ Podemos centralizar a imagem utilizando as regras de background position com o valor "center".📐 Também podemos utilizar a regra de "cover" para exibir a imagem completa, mesmo que seja retangular.⚙️ Essas técnicas são úteis para projetos profissionais no futuro.Pontos principais🖼️ Centralizar a imagem de background melhora a visualização e aproveita o espaço útil da tela.⚙️ Podemos utilizar as regras de background position com valor "center" para centralizar a imagem.📐 Também podemos usar a regra de "cover" para exibir a imagem completa, mesmo que seja retangular.🖼️ A centralização e o uso do "cover" são úteis para projetos profissionais no futuro.📐 A centralização é feita posicionando o elemento do HTML do topo para a esquerda.🖼️ A regra de "cover" exibe o máximo possível da imagem, mesmo que seja retangular.📐 O formato da imagem pode influenciar na visualização, mas o objetivo é mostrar todo o contexto da imagem.
